Jammming Web-App

Description

Jammming is an app to create custom playlist and save them to Spotify. The main functionality is searching for songs by title, artist or album title using the Spotify API. From the results-list users can add songs to their playlists. No duplicates could be added. When the playlist is complete it can be added to the users Spotify account, again using the API.

Languages used:

React, Javascript, CSS, HTML
